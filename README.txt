Contenido del Repositorio PAPER-SENSORS:

Dentro del repositorio se encuentran los archivos de código fuente:

• filter.py : Código fuente del programa encargado de extraer la información de uno o varios archivos JSON obtenidos con cuckoo

• gui.py: Código fuente de la interfaz gráfica que invoca los métodos del archivo filter.py

◦ ml_predictor.py: Script en python que puede ser empleado para clasificar nuevos artefactos.

◦ dl_predictor.py: Script en python que puede ser empleado para clasificar nuevos artefactos.
